## 📊 **Gemini API 키 발급 방법**

1. **Google AI Studio 접속**
   - https://aistudio.google.com/app/apikey

2. **API 키 생성**
   - "Create API Key" 클릭
   - 프로젝트 선택 (또는 새로 생성)
   - API 키 복사

3. **환경변수 설정**
   ```bash
   # .dev.vars 파일에 추가
   GEMINI_API_KEY=your-api-key-here
   ```

4. **Cloudflare 배포 시**
   ```bash
   # Cloudflare Dashboard → Settings → Environment variables
   GEMINI_API_KEY=your-api-key-here
   ```
