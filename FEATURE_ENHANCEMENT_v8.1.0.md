#### **환불 정책 상세 모달**

**환불 가능 조건**:
- 구매일로부터 7일 이내
- 크레딧 미사용 또는 30% 이하 사용
- 서비스 오류로 인한 크레딧 차감 시
- 중복 결제 발생 시

**환불 불가 조건**:
- 구매일로부터 7일 경과 후
- 크레딧 30% 초과 사용 시
- 콘텐츠를 다운로드하거나 복사한 경우
- 무료 체험 크레딧

**환불 금액 계산표**:
| 사용 크레딧 | 환불 비율 | 예시 (10,000원 구매 시) |
|-------------|-----------|------------------------|
| 0% (미사용) | 100% | 10,000원 환불 |
| 1~10% | 90% | 9,000원 환불 |
| 11~20% | 80% | 8,000원 환불 |
| 21~30% | 70% | 7,000원 환불 |
| 31% 이상 | 환불 불가 | - |

**환불 처리 기간**:
- 신용카드: 3~7 영업일
- 계좌이체: 1~3 영업일
- 간편결제: 각 PG사 정책에 따름

**환불 요청 방법**:
- 이메일: refund@haruhanpo.com
- 고객센터: 1234-5678 (평일 09:00~18:00)
- 필수 정보: 주문번호, 구매일자, 환불 사유

### ✅ **2. 크레딧 관리 개선**

#### **헤더 크레딧 표시 개선**

**이전**:
```
무료회원 | 3회 남음
```

**현재**:
```
무료회원 | 3 크레딧 [ℹ️] [+ 충전]
```

**추가된 기능**:
1. **크레딧 상세 정보 아이콘** (ℹ️)
   - 클릭 시 크레딧 잔액 및 사용 내역 표시
   - 현재: alert으로 구현 (추후 모달로 개선 가능)

2. **크레딧 충전 버튼** (+ 충전)
   - 클릭 시 충전 옵션 안내
   - 충전 옵션:
     - 10 크레딧: 10,000원
     - 50 크레딧: 45,000원 (10% 할인)
     - 100 크레딧: 80,000원 (20% 할인)
   - 현재: confirm/alert으로 구현 (추후 결제 시스템 연동 가능)

#### **크레딧 정책 명시**
- 콘텐츠 1개 생성 = 1 크레딧
- 크레딧 유효기간: 구매일로부터 1년
- 미사용 크레딧 환불 가능 (7일 이내)
