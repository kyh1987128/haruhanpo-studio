## 🧪 테스트 시나리오

### 1. 기본 기능 테스트

#### 1.1 이미지 업로드
**테스트 케이스:**
- [ ] 클릭하여 이미지 선택 (1장)
- [ ] 드래그 앤 드롭으로 이미지 업로드 (여러 장)
- [ ] 이미지 미리보기 표시 확인
- [ ] 이미지 개별 삭제 (X 버튼)
- [ ] 최대 10장 제한 테스트
- [ ] 50MB 용량 제한 테스트
- [ ] 이미지 없이 생성 시도 (에러 메시지 확인)

**예상 결과:**
- 이미지 업로드 시 즉시 미리보기 표시
- 10장 초과 시: "최대 10장까지 업로드 가능합니다." 알림
- 50MB 초과 시: "전체 이미지 크기는 50MB를 초과할 수 없습니다." 알림
- 이미지 없이 생성 시: "최소 1장의 이미지를 업로드해주세요." 알림

#### 1.2 브랜드 정보 입력
**테스트 케이스:**

**필수 필드:**
- [ ] 브랜드명 입력
- [ ] 핵심 키워드 입력 (쉼표로 구분)
- [ ] 빈 값으로 제출 시도 (에러 확인)

**선택 필드:**
- [ ] 회사 상호명
- [ ] 사업자 유형 (개인/법인/프리랜서)
- [ ] 지역 (17개 시도 확인)
- [ ] 타겟 성별 (전체/남성/여성)
- [ ] 연락처
- [ ] 웹사이트 URL
- [ ] SNS 계정

**타겟팅 설정:**
- [ ] 톤앤매너 (캐주얼/전문가/감성)
- [ ] 타겟 연령대 (10대~50대+)
- [ ] 산업 분야 (6개 옵션 확인)

**예상 결과:**
- 모든 필드가 정상적으로 입력됨
- 필수 필드 누락 시 적절한 에러 메시지
- 드롭다운 선택지가 정확히 표시됨

#### 1.3 플랫폼 선택
**테스트 케이스:**
- [ ] 블로그만 선택
- [ ] 인스타그램만 선택
- [ ] 스레드만 선택
- [ ] 유튜브만 선택
- [ ] 전체 4개 선택
- [ ] 아무것도 선택 안 함 (에러 확인)

**예상 결과:**
- 최소 1개 선택 필수
- 선택 안 함 시: "최소 1개 플랫폼을 선택해주세요." 알림

#### 1.4 콘텐츠 생성
**테스트 케이스:**
- [ ] 이미지 1장 + 블로그만
- [ ] 이미지 2장 + 블로그+인스타
- [ ] 이미지 4장 + 전체 플랫폼
- [ ] 이미지 10장 + 전체 플랫폼

**예상 결과:**
- 로딩 스피너 표시 (30-60초)
- 생성 완료 후 성공 알림
- 선택한 플랫폼만 탭에 표시
- 각 탭에 콘텐츠 정상 표시

#### 1.5 결과 확인 및 관리
**테스트 케이스:**
- [ ] 탭 전환 (블로그 → 인스타 → 스레드 → 유튜브)
- [ ] 글자 수 카운터 확인
- [ ] 복사 버튼 클릭
- [ ] 다운로드 버튼 클릭 (.txt 파일 확인)
- [ ] 다운로드 파일명 형식 확인 (브랜드명_플랫폼_날짜.txt)

**예상 결과:**
- 탭 전환 즉시 반영
- 글자 수 정확히 표시
- 복사 시 클립보드에 저장, "클립보드에 복사되었습니다!" 알림
- 다운로드 정상 작동

### 2. 고급 기능 테스트

#### 2.1 프로필 관리
**테스트 케이스:**

**프로필 저장:**
- [ ] 모든 필드 입력 후 "프로필 저장" 클릭
- [ ] 프로필 이름 입력 (예: "테스트 브랜드 1")
- [ ] 저장 완료 알림 확인
- [ ] 여러 개 프로필 저장 (최소 3개)

**프로필 불러오기:**
- [ ] "프로필 불러오기" 클릭
- [ ] 모달에 저장된 프로필 목록 확인
- [ ] 프로필 선택하여 불러오기
- [ ] 모든 필드가 자동 채워지는지 확인

**프로필 삭제:**
- [ ] 프로필 목록에서 삭제 버튼 (휴지통 아이콘) 클릭
- [ ] 확인 다이얼로그 표시 확인
- [ ] 삭제 후 목록에서 제거 확인

**예상 결과:**
- 프로필 저장: "프로필이 저장되었습니다!" 알림
- 프로필 적용: "프로필이 적용되었습니다!" 알림, 모든 필드 자동 채움
- 프로필 삭제: "프로필이 삭제되었습니다." 알림

#### 2.2 콘텐츠 히스토리
**테스트 케이스:**

**자동 저장:**
- [ ] 콘텐츠 생성 후 자동 저장 확인
- [ ] "히스토리" 버튼 클릭하여 목록 확인

**히스토리 조회:**
- [ ] 날짜별 생성 목록 표시 확인
- [ ] 브랜드명, 생성 시간, 플랫폼 태그 표시 확인

**히스토리 불러오기:**
- [ ] 히스토리 항목 클릭 (눈 아이콘)
- [ ] 과거 생성된 콘텐츠 재표시 확인

**히스토리 삭제:**
- [ ] 히스토리 항목 삭제 (휴지통 아이콘)
- [ ] 확인 후 목록에서 제거 확인

**예상 결과:**
- 히스토리 자동 저장 (최대 50개)
- "히스토리가 로드되었습니다!" 알림
- "히스토리가 삭제되었습니다." 알림

#### 2.3 비용 계산기
**테스트 케이스:**

**실시간 계산:**
- [ ] 이미지 1장 업로드 → 비용 업데이트 확인
- [ ] 이미지 추가 → 비용 증가 확인
- [ ] 이미지 삭제 → 비용 감소 확인
- [ ] 플랫폼 체크/해제 → 비용 변경 확인

**표시 내용:**
- [ ] 달러 금액 표시
- [ ] 원화 금액 표시 (≈ 기호 포함)
- [ ] 환율 정보 표시
- [ ] 이미지 개수 × 단가
- [ ] 플랫폼 개수

**예상 결과:**
- 실시간 업데이트
- 달러: 큰 파란색 텍스트
- 원화: 중간 보라색 텍스트

### 시나리오 1: 신규 사용자 - 첫 콘텐츠 생성
```
1. 페이지 접속
2. 이미지 2장 드래그앤드롭
3. 브랜드명 입력: "테스트 카페"
4. 키워드 입력: "커피, 디저트, 강남"
5. 지역 선택: "서울"
6. 톤앤매너: "캐주얼"
7. 타겟 연령대: "20대"
8. 산업 분야: "F&B"
9. 플랫폼 선택: 블로그, 인스타그램
10. 비용 확인: $0.09 ≈ ₩117
11. "콘텐츠 생성하기" 클릭
12. 로딩 대기 (30-60초)
13. 결과 확인:
    - 블로그 탭: 1,300자 이상 콘텐츠
    - 인스타그램 탭: 캡션 + 해시태그 25-30개
14. 블로그 복사 → 클립보드 확인
15. 인스타그램 다운로드 → 파일 확인
```

**예상 소요 시간:** 3-5분  
**예상 비용:** $0.09 (약 ₩117)
